# Multivariable Derivative Calculator

## Description
Takes a list of numbers of degree n for example  
((1 2)(2 3)(3 4)) is (1+2y)+(2+3y)x+(3+4y)x^2  
and returns the derivative in terms of a list of numbers of degree n.
